# Zabbix Template - UPS LogBlueWeb SNMP

Template para monitoramento de nobreaks compatíveis com LogBlueWeb - UPS Agent via SNMP.

## Recursos Monitorados

- Status do Nobreak
- Percentual de Bateria
- Carga Atual
- Temperatura
- Tensão de Entrada
- Tensão de Saída
- Frequência de Entrada
- Falhas de Energia
- Disponibilidade ICMP
- Disponibilidade SNMP

## Triggers Incluídas

- Bateria Baixa
- Sobrecarga
- Falta de Energia
- Temperatura Alta
- Temperatura Crítica
- Frequência Fora do Padrão
- Indisponibilidade SNMP
- Indisponibilidade ICMP

## Requisitos

- Zabbix 7.x
- SNMP habilitado no LogBlueWeb UPS Agent
